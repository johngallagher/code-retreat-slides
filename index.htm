The point of a code retreat is not to make progress on the problem.

The point is to experiment without pressure, and write good code!

Format of each session

  1. 45 minutes coding
  2. 10 minutes retrospective
  3. 5 minutes break

Ping-Pong Programming

  1. Joe writes a new test and sees that it fails
